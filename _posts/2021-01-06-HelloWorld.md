## Blog features.

This blog is powered by Jekyll, a static website generator made with Ruby - a programming language. Because of this, I use Markdown to write the articles, and so I can use code blocks and math expressions.

For instance:

We can say that the derivative global definition can be defined by

$$ \dfrac{dy}{dx} = \lim_{h \to 0} \dfrac{f(x+h)- f(x)}{h}$$

Or solve a short problem:

$$
\begin{aligned}
\lim_{x \to 2} \dfrac{x^2 - 4}{x - 2} &= \lim_{x \to 2} \dfrac{(x+2)(x-2)}{x-2} \\\\
&= \lim_{x \to 2} x+2\\\\
&= 4
\end{aligned}
$$

And you can see the code (highlighted) of this math in \\(\LaTeX\\):
